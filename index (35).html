🗄️ Base de Dados Partilhada (JSONBin)
Para partilhar dados entre utilizadores, configure um bin gratuito em jsonbin.io. Crie conta gratuita → New Bin → copie o BIN ID e a Master Key.

ℹ️ Como criar um JSONBin gratuito:
1. Aceda a jsonbin.io e crie conta gratuita
2. Clique em "+ New Bin" e cole {"records":[],"index":[]}
3. Copie o BIN ID do URL
4. Em API Keys crie ou copie a Master Key
5. Cole ambos acima e clique "Guardar Configurações"
